# W3Schools (w3schools.com) — Domain Skill

## Overview
W3Schools is a beginner-friendly web development tutorial site. Pages are organized by language/topic, with a left sidebar for navigation, a main content area with explanations and code examples, and a "Try it Yourself" editor.

## Navigation Tips
- The left sidebar lists all tutorial topics for the current language — use it to jump between lessons.
- The top navigation bar switches between languages (HTML, CSS, JavaScript, Python, SQL, Java, etc.).
- Each tutorial page has "Previous" and "Next" buttons at the bottom to move sequentially through lessons.
- The search bar at the top right is the fastest way to find a specific topic or function.

## Content Structure
- Each page follows a consistent pattern: concept explanation → simple example → "Try it Yourself" button → additional details.
- Code examples are displayed in styled boxes. The green "Try it Yourself" button opens an in-browser editor.
- Reference pages (e.g., SQL Functions, CSS Properties) are separate from tutorial pages and list all options for a topic.

## Reading & Extracting Content
- Use `get_page_text` to efficiently read tutorial content without scrolling.
- The main article content is in the `#main` div — focus there to avoid sidebar noise.
- Code examples are in `<div class="w3-code">` or `<pre>` blocks.

## SQL Tutorial Specifics (current page)
- The SQL section covers SELECT, INSERT, UPDATE, DELETE, JOINs, functions, and more.
- The left sidebar groups SQL topics: SQL Basics → SQL Database → SQL References.
- Exercises and quizzes are available at the bottom of many pages for practice.
